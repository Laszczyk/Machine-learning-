# Machine Learning – Project Portfolio

This repository contains a series of laboratory projects covering fundamental and advanced machine learning techniques, from data preprocessing to ensemble methods and model optimization.

## Project Overview

| Project | Description | Report Preview (HTML) |
| :--- | :--- | :--- |

| **2. Data Preprocessing** | Advanced techniques for handling **missing data** (imputation). Analysis of the impact of different cleaning methods on model accuracy using the Airline dataset. | [View Report](https://htmlpreview.github.io/?https://github.com/Laszczyk/Machine-learning-/blob/main/Miczek_Laszczyk_UM2/Miczek_Laszczyk_UM2/Miczek_Laszczyk_UM2.html) |
| **3. Ensemble Methods: Bagging** | Comparison between simple **Decision Trees** and **Bagging** techniques. ## Skills & Methodologies
* **Algorithms:** KNN, Logistic Regression, SVM, Decision Trees, Random Forest, Bagging, Boosting.
* **Evaluation Metrics:** Accuracy, Precision, Recall, F1-score, ROC/AUC, Specificity/Sensitivity.
* **Preprocessing:** Imputation of missing values, data normalization, handling imbalanced classes.
* **Tools:** R / RMarkdown.
